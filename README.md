## 🚀 Overview

Skill2Role is a modern web application that helps users discover career roles based on their interests and skills. It uses a logic-driven matching system to rank roles and highlight skill gaps, while also allowing real-world exploration through search and learning resources.

---

## ✨ Features

* 🎯 Multi-interest selection
* 📊 Skill-based role matching with percentage scoring
* 📉 Skill gap analysis (missing skills + progress)
* 🔍 Direct Google search integration
* 🎥 “Watch” option for learning
* ⭐ Bookmark system (saved roles using localStorage)
* 🎨 Multiple UI themes (Light, Dark, Soft Blue, Midnight, Minimal Gray)
* 🧠 Large structured dataset of roles and skills

## 🧠 Core Logic

* Interest → Roles mapping
* Roles → Skills mapping
* Matching formula:

match % = (matched skills / total required skills) × 100

* Roles are ranked by highest match
